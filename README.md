# Agentic AI Insurance Claim Investigation System

An advanced Agentic AI system that analyzes insurance claims using multiple AI agents to detect fraud, evaluate risk, predict claim probability, and recommend decisions.

The system uses *Machine Learning + Graph AI + Multi-Agent Architecture to automate insurance claim investigation.
# Architecture

User Claim Input  
↓  
Agent Controller
↓  
Claim Prediction Agent  
Fraud Detection Agent  
Fraud Network Detection Agent  
Risk Evaluation Agent  
Pricing Agent  
↓  
Decision Agent  
↓  
Investigation Report  
↓  
Stored in Vector Memory

---
# Feature

- Multi-Agent AI Architecture
- Machine Learning based predictions
- Fraud detection using anomaly detection
- Fraud network analysis using Graph AI
- Risk clustering
- Premium pricing prediction
- Investigation report generation
- Vector memory storage
- File upload support
- Automatic dummy dataset generation
